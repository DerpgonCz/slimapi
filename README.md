# SlimAPI
A collection of tools for building APIs. Based on [Slim Framework][link-slim].

## Installation
``` bash
$ composer require slimapi/slimapi
```

## Development
```bash
$ docker-compose up --always-recreate-deps --build --force-recreate
```
